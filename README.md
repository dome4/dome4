# Resume of Dominic Seitz

## Development

1. Serve development server via `bundle exec jekyll serve`
2. Browse to [http://localhost:4000](http://localhost:4000)

## Deployment

- `bundle exec jekyll build`
- deploy generated content of folder `_site`
